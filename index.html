Git & Github

O que é o Git?

O Git é um sistema de controlo de versão (Version Control System), como diz o nome é usado para gravar alterações feitas no código ao longo do tempo em uma base de dados especial chamado repositório.

O que é p Github?

O Github por outro lado é o Git hospedado em uma nuvem ou via web, é usado para conectar os diferentes utilizadores de uma equipe e não só.
